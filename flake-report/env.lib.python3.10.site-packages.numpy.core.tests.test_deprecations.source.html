55 """Test if DeprecationWarnings are given and raised.
56  
57 This first checks if the function when called gives `num`
58 DeprecationWarnings, after that it tries to raise these
59 DeprecationWarnings and compares them with `exceptions`.
60 The exceptions can be different for cases where this code path
61 is simply not anticipated and the exception is replaced.
62  
63 Parameters
64 ----------
65 function : callable
66 The function to test
67 num : int
68 Number of DeprecationWarnings to expect. This should normally be 1.
69 ignore_others : bool
70 Whether warnings of the wrong type should be ignored (note that
71 the message is not checked)
72 function_fails : bool
73 If the function would normally fail, setting this will check for
74 warnings inside a try/except block.
75 exceptions : Exception or tuple of Exceptions
76 Exception to expect when turning the warnings into an error.
77 The default checks for DeprecationWarnings. If exceptions is
78 empty the function is expected to run successfully.
79 args : tuple
80 Arguments for `function`
81 kwargs : dict
82 Keyword arguments for `function`
